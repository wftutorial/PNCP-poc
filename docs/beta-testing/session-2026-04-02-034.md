### Problema estrutural identificado (ISSUE-065)
O setor "Transporte e Veículos" tem escopo **excessivamente amplo**. Para Ricardo (transporte escolar de pessoas), apenas **locação/fretamento** e **transporte escolar** são relevantes. Mas o setor inclui:
- **Compra de veículos** (21 FPs / 38 = 55% dos FPs) — ambulâncias, hatch, SUV, van, pick-up, caminhão
- **Combustíveis** (4 FPs) — etanol, gasolina, diesel para frota municipal
- **Manutenção mecânica** (7 FPs) — funilaria, hidráulica, peças, viaturas
- **Insumos automotivos** (4 FPs) — óleos, filtros, pneus, baterias
- **Seguros** (1 FP) — cobertura securitária
- **Outros** (1 FP) — sonorização, lavagem

### Root Cause Analysis
O filtro de keywords do setor Transporte inclui termos como "veículo", "automóvel", "van", "ônibus" que capturam tanto locação/fretamento (TP) quanto compra de veículos novos (FP). As exclusões adicionadas no commit (0km, combustível, peças) não foram suficientes porque:
1. Compra de veículos sem "0km" ou "zero km" no título (ex: "Aquisição de veículo tipo van") não é capturada
2. "Manutenção veicular" não foi excluída
3. "Aquisição de veículo" como exclusão poderia funcionar mas provavelmente quebraria TPs

### Fix recomendado — Abordagem estrutural
O setor Transporte precisa de **sub-categorização** ou **exclusões massivas**:

**Opção 1 (cirúrgica — exclusões):**
```yaml
# sectors_data.yaml, setor Transporte, exclusions:
# Compra de veículos (o maior bloco de FPs):
- "aquisição de veículo"
- "aquisicao de veiculo"
- "aquisição de automóvel"
- "aquisicao de automovel"
- "aquisição de ambulância"
- "aquisicao de ambulancia"
- "aquisição de caminhão"
- "aquisicao de caminhao"
# Manutenção mecânica:
- "manutenção veicular"
- "manutencao veicular"
- "manutenção preventiva e corretiva"
- "manutencao preventiva e corretiva"
- "funilaria"
- "lanternagem"
# Insumos automotivos:
- "lubrificantes automotivos"
- "óleos lubrificantes"
- "oleos lubrificantes"
- "filtros automotivos"
- "pneus e protetores"
# Seguros:
- "cobertura securitária"
- "cobertura securitaria"
# Lavagem:
- "lavagem de veículos"
- "lavagem de veiculos"
- "higienização de veículos"
- "higienizacao de veiculos"
# Outros:
- "sonorização"
- "motoniveladora"
```

**Opção 2 (estrutural — restringir keywords):**
Remover keywords genéricas como "veículo", "automóvel" e manter apenas "transporte escolar", "locação de veículos", "fretamento", "ônibus escolar", etc.

**ATENÇÃO:** Opção 1 é mais segura (não remove TPs), mas o volume de exclusões é grande. Opção 2 é mais limpa mas pode perder TPs legítimos.
