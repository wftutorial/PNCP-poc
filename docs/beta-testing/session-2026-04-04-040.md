### P2 — Software: Sub-categorização desenvolvimento vs revenda
**Problema:** Setor "Software e Sistemas" mistura dois negócios completamente distintos:
- Empresas de desenvolvimento de software (software houses, SaaS)
- Revendedores de licenças (Microsoft, Autodesk, VMware CSPs)

Juliana (software house) recebe 4/9 resultados irrelevantes que são para revendedores. FPs observados: Microsoft 365 subscription, Autodesk AEC Collection, Hipervisor VMware, firewall locação.

**Solução recomendada: Split em 2 setores — análogo ao split transporte_servicos / frota_veiculos (ISSUE-064/065)**

---

#### Plano de implementação: split `software` → `software_desenvolvimento` + `software_licencas`

**Analogia com transporte:**
| Transporte (antes) | Software (antes) |
|---|---|
| Serviços contratados de transporte + Compra de frota = 1 setor com FPs | Desenvolvimento/SaaS + Revenda de licenças = 1 setor com FPs |
| Split: `transporte_servicos` + `frota_veiculos` | Split: `software_desenvolvimento` + `software_licencas` |
