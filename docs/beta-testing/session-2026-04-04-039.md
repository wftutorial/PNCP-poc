## Teste E2E: Stripe Checkout em Producao (Live Mode)

### Contexto

O checkout Stripe era o ultimo fluxo critico sem verificacao live em producao. Sessoes anteriores (037-038) confirmaram o layout de /planos e o redirect para Stripe, mas nao completaram o checkout E2E porque:
- Producao usa chave live (`sk_live_`), cartoes de teste sao rejeitados
- Nenhum cupom 100% existia para testar sem custo

### Metodologia

1. Criar cupom 100% off no Stripe Dashboard via Playwright
2. Iniciar checkout no SmartLic producao via Playwright
3. Aplicar cupom para total R$ 0,00
4. Completar assinatura
5. Verificar webhook processing + ativacao
6. Cancelar assinatura de teste
7. Excluir cupom
