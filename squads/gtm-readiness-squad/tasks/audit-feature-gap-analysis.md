# *audit-gaps

Identify missing table-stakes features vs market.

## Steps

1. List all SmartLic features
2. List competitor features (especially Siga Pregao, ConLicitacao)
3. Identify table-stakes gaps (alerts, docs, mobile)
4. Verify differentiators are working (AI viability, multi-source)
5. Prioritize gaps by impact on conversion

## Output

Score (0-10) + gap matrix + priority recommendations
