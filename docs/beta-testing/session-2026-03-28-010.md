## Issues Novas Nesta Sessao

### ISSUE-026: Resumo IA fabrica relevancia — diz que resultados sao sobre o termo quando nao sao

**Persona:** Alex "The Closer"
**Severidade:** P1 — Critical
**Area:** Buscar / Resumo IA / LLM
**Data:** 2026-03-28 16:35 UTC
**Status:** ABERTO

#### Passos para Reproduzir
1. Login em https://smartlic.tech (admin)
2. Navegar para /buscar
3. Modo "Termos Especificos" > digitar "uniformes escolares" > Enter
4. SP selecionado, clicar "Buscar 1 termo"
5. Aguardar resultados

#### Comportamento Observado
- O resumo IA diz: **"Foram identificadas 10 licitacoes relacionadas a uniformes escolares no estado de Sao Paulo"**
- Na realidade, os 10 resultados sao:
  1. Biodescontaminacao COVID-19/H1N1 (SP Franca)
  2. Sistema web gestao publica municipal (SP Araraquara)
  3. Carnes, massas e iogurte para SEDUC (SP Sao Vicente)
  4. Manutencao rede eletrica transformadores (SP Sao Jose Rio Pardo)
  5. Bombas submersas pocos artesianos (SP Sao Jose Rio Preto)
  6. Projeto metro linha 22-marrom lote 1 (SP Sao Paulo)
  7. Projeto metro linha 22-marrom lote 2 (SP Sao Paulo)
  8. Projetos engenharia corredores transporte (SP Santo Andre)
  9. Materiais didaticos afro-brasileira/indigena (SP Mogi das Cruzes)
  10. Generos alimenticios hortifruti (SP Mairinque)
- **0/10 sao sobre uniformes escolares**

#### Comportamento Esperado
- O resumo IA deveria refletir HONESTAMENTE o conteudo dos resultados
- Se nenhum resultado e sobre uniformes, o resumo deveria dizer isso
- Fabricar relevancia destrói a confianca do usuario na plataforma

#### Screenshot
`docs/beta-testing/screenshots/session-010/ISSUE-017-uniformes-escolares-0-relevant.png`

#### Notas Tecnicas
- O resumo e gerado pelo GPT-4.1-nano via ARQ background job
- O prompt provavelmente recebe os resultados + o termo de busca e assume que sao relevantes
- O prompt precisa ser reformulado para AVALIAR a relevancia antes de afirmar
- Relacionado a ISSUE-017 — se os resultados sao irrelevantes, o resumo amplifica o problema

#### Fix Sugerido
- Adicionar ao prompt do resumo uma instrucao para verificar se os resultados realmente correspondem aos termos buscados
- Se nao correspondem, gerar um aviso honesto ao inves de fabricar relevancia
