# *audit-acquisition

Assess readiness for user acquisition channels.

## Steps

1. Check SEO: meta tags, sitemap.xml, robots.txt
2. Check analytics: Mixpanel events, UTM support
3. Check email: welcome sequence, templates
4. Assess conversion funnel tracking
5. Evaluate channel readiness (organic, paid, social, referral)

## Output

Score (0-10) + channel readiness matrix + recommendations
